  == Introduction
  It's POC project demostrating the feasibility to implement quikly a service for save and read an account user.
  The account user is an entity with : 
   -User Name,
   -Birthdate,
   -Country,
   -phone Number
   -Gender
  
  This project use :
  - Spring Boot
  - Spring Data
  - Validator
  - Lombock
  - Swagger
  - H2 embeded database
  - maven
  
   
  
  
  == Prerequis 
  - JDK11 
  - Maven 3
  - IDE (Intelij or Eclipse with lombok plugin)
